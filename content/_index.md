# *L*ight *E*nigmatic *V*oid Linux
is a Void-based Linux distribution that focuses on functionality and
minimalism.

|                      |               |
| -------------------- | ------------- |
| **C Library**        | *musl*        |
| **Window manager**   | *DWM*         |
| **Status bar**       | *DWM Blocks*  |
| **Terminal**         | *ST*          |
| **Shell**            | *FISH*        |
| **Lock screen**      | *SLock*       |
| **Browser**          | *Firefox*     |
| **Document reader**  | *Zathura*     |
| **File manager**     | *NNN*         |
| **Image viewer**     | *SXIV*        |
| **Video player**     | *MPV*         |
| **Password manager** | *pass*        |
| **Email client**     | *NeoMutt*     |
| **Text editor**      | *NeoVIM*      |
